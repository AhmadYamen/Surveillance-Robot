# Surveillance Robot

## Project Structure

```
/                  # Project root
├── docs/         # Documentation files
├── src/          # Source files
│   ├── embedding_engine.py
│   ├── image_modification.py
│   ├── reco_engine.py
│   ├── training_app.py
│   ├── config.yaml
│   └── models/    # Model files
├── tests/        # Test files
│   ├── __init__.py
│   └── test_placeholder.py
├── LICENSE       # Project License
├── README.md     # Project Documentation
└── setup.py      # Packaging configuration
```

## Usage

1. Clone the repository.
2. Install dependencies.
3. Run the training application: `python src/training_app.py`

## Tests

Use the `tests/` directory for unit tests.

## Documentation

Visit the `docs/` directory for further documentation and updates.
